# StudentDB SQL Join Project

## Introduction

This project is based on SQL database named **StudentDB**.
It is created to learn and perform different types of **SQL Join queries**.

## Database Name

StudentDB

## Tables Created

### 1. Customer1

This table stores basic customer information:

* CustomerID
* CustomerName
* City

### 2. CustomerDetails

This table stores customer extra details:

* DetailsID
* CustomerID
* Phone
* Email

## Join Queries Used

### Inner Join

Used to show matching records from both tables.

### Left Join

Shows all records from Customer1 and matching from CustomerDetails.

### Right Join

Shows all records from CustomerDetails and matching from Customer1.

### Full Join

Shows all records from both tables.

## Purpose

The purpose of this project is to understand:

* SQL joins
* Relationship between tables
* Advanced SQL queries
